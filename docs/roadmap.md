# Roadmap

LL13 is intentionally small and practical. The roadmap favors features that help people learn Laravel by building with Laravel.

## Now

- Study-first learning paths.
- MCQ and task checkpoints.
- Explanations after completed MCQs.
- Learning logs and progress metrics.
- Public documentation and project branding.

## Next

- More curated Laravel 13 quests.
- Better checkpoint filtering and search.
- Difficulty-based review mode.
- Exportable progress summary.
- More screenshots and examples in the docs.

## Later

- Spaced repetition for missed questions.
- Optional project templates for hands-on tasks.
- Richer admin tooling for authoring quests.
- Better accessibility review and keyboard workflows.
- Optional hosted demo environment.

## Not Planned Right Now

- Formal certification.
- Paid course platform features.
- External user tracking.
- Vendor-specific deployment lock-in.
